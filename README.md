# Survey-Facebook
Posting form survei Anda di Facebook  Anda sudah membuat form survei mengenai keingintahuan Anda mengenai feedback atau tanggapan teman atau fans Anda terhadap bisnis atau event yang akan Anda buat. Sekarang Anda ingin menyebarkan form survei Anda tersebut ke Facebook dengan format yang mudah digunakan. Jangan khawatir, dengan SurveyMonkey Anda bisa menyebarkan form survei Anda dengan mudah langsung ke personal page Anda menggunakan Facebook Collector atau mem-posting daftar pertanyaan ke fan page Anda menggunakan Web Link Collector.
